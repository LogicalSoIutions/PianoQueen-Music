# PianoQueen

A RuneLite sound plugin that plays a configurable sound when you enter your player-owned house.

PianoQueen downloads WAV files from the `sounds` branch of
https://github.com/LogicalSoIutions/PianoQueen-Music into `.runelite/piano-queen/`.
Selected songs are downloaded on demand and reused from the local cache.

Audio playback is based on the BSD-2 licensed Music Replacer plugin approach, using AudioCue for WAV files.
